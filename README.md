# Linker-IM

![golang](https://img.shields.io/badge/golang-%E2%89%A51.11-blue.svg)

***(In development.)***

Instant message push server. 



#### Build

```bash
git clone https://github.com/Sunmxt/linker-im.git
cd linker-im
```

*Build from **source**:*

```bash
make
```

Or build **docker image**:

```bash
docker build . -t linker-im
```
